Team Explorer Everywhere (version 10.0.0.201003261741)

 label command:

  Attaches a label to or removes a label from a version of a file or folder in the source control server.

 Valid option sets:

  label [-owner:<value>] [-version:<value>] [-comment:<value>|@valuefile] [-child:fail|replace|merge] [-recursive]
        <labelName>[@<scope>] <itemSpec>...
  label -delete [-owner:<value>] [-version:<value>] <labelName>[@<scope>]
