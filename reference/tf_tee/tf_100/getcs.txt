Team Explorer Everywhere (version 10.0.0.201003261741)

 getcs command:

  Gets only the files in a particular changeset at the version of the changeset.  Only files in folders mapped to
  working folders are retrieved.
  
  If the latest option is present, the latest versions of the files in the specified changeset are retrieved
  instead of the versions that were created by the changeset.

 Command aliases: getcs getchangeset

 Valid option sets:

  getcs -changeset:<value> [-latest]
