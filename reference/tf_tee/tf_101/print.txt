Team Explorer Everywhere (version 10.1.0.201101271439)

 print command:

  Retrieves a copy of one file from the source control server and prints its contents to the standard output stream.
  
  The version to retrieve may be specified through the 'version' option or as a version specification suffix to the item specification (example:
  '$/file.txt;C34').

 Command aliases: cat print

 Valid option sets:

  print [-version:<value>] <itemSpec>
