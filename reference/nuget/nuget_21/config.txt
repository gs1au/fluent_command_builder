usage: NuGet config -Set name=value

Gets or sets NuGet config values.

options:

 -Set +                                              One on more key-value pairs to b
                           e set in config.
 -Help                (?)                            help
 -Verbosity                                          Display this amount of details i
                           n the output: normal, quiet, det
                           ailed.
 -NonInteractive                                     Do not prompt for user input or 
                           confirmations.

For more information, visit http://docs.nuget.org/docs/reference/command-line-reference
